# Zaheer Unfiltered

A premium 18+ content platform built with HTML, CSS, and JavaScript.

## Features

- **Age Verification Gate**: Ensures compliance with age restrictions.
- **Premium UI**: Dark theme, Netflix-style content sliders, and responsive design.
- **Dynamic Content**: Mock data rendering for videos and categories.
- **Single Page Application (SPA) Feel**: Smooth transitions between Home and Explore views without page reloads.

## Deployment

This project is ready for GitHub Pages.

1.  Push the code to a GitHub repository.
2.  Go to **Settings** > **Pages**.
3.  Select the `main` branch and `/ (root)` folder.
4.  Save.

## Local Development

Simply open `index.html` in your browser or use a live server extension.
